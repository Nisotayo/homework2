CoWnalysisNN Research Project

I am the lead researcher responsible for development of the data analysis pipeline at the core of this project. The following is a high level overview of the pipeline:

First a CT or MRI scan is uploaded to the storage server through the frontend webserver. The scan is then uploaded to Monsoon so high performance GPU resources can be utilized. Inference is performed on the scan using a custom neural network, which produces a segmentation mask that labels the different arteries. The segmentation mask is then processed with surface meshing, artery centerline extraction, and centerline angle extraction. Finally the data is sent back to the storage server where it is added to a database. Currently the data from over 5000 scans is being used to train a Bayesian neural network which will be our predictive model.
